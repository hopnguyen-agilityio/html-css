## How to run
- Clone source:

```
 git clone https://github.com/ngoclien08/html-css-training.git
```

- Move to project branch

```
git checkout feature/practice-two
```

- Next open folder practice-two

```
cd practice-two
```

- Install packages

```
npm install
```

- Run project

```
npx parcel src/index.html
```

- Build project

```
npm run build
```
